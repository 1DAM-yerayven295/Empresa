# Norteoliva
**Modelo de negocio:** Mixto

## ¿Por qué elegí la GPLv3?

Porque obliga a que cualquier modificación del núcleo también se comparta bajo GPLv3. Así nadie se lleva mi trabajo, le pone un logo y lo vende como si lo hubiera hecho él.  
Y lo mejor: puedo ofrecer módulos de análisis o exportación como servicios privados, que se conectan al núcleo pero no forman parte de él. Eso me permite cobrar por lo que realmente vale —sin romper la licencia.

## ¿Qué derechos das a otros?

- Usar el software libremente.  
- Estudiar cómo funciona.  
- Compartirlo con otros.  
- Modificarlo para adaptarlo a su cooperativa.

## ¿Qué limitaciones impones?

Cualquier persona que redistribuya el código fuente —o una versión modificada del núcleo— **debe hacerlo bajo la misma licencia GPLv3**.  
Eso significa: no puedes coger mi código, hacerle cambios y venderlo como software cerrado. Si lo das fuera, tiene que seguir siendo abierto.
